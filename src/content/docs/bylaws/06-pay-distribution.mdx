## <small>PAY-01</small> 收入定義與案子分類 {#pay-01}

本社參與案子依資金來源與合作方式分為四類：

:::note[為什麼要區分四種案子？]
不同案子類型反映了不同的資金來源、風險承擔與回報機制。委託案與過案體現了本社對外部關係的不同角色（直接承包人 vs. 行政支持方）；
自籌案與補助案則反映本社內生的創新動力。清晰的分類有助於透明的財務管理與公平的資源分配。
:::

## <small>PAY-01A</small> 總收入基準 {#pay-01a}

委託案、補助案與過案之收入以 **總收入（Gross Revenue）** 計算，不先扣除成本、稅費或雜支。以下將在後續條款中規定成本提撥、稅費處理與分配原則，但基礎比較與提撥計算皆以總收入為基準，以確保透明與公平。

:::note[為什麼以總收入為基準？]
以總收入為基準可以避免因成本結構不同而產生的計算不透明或爭議。成本的定義與範圍可能因專案性質、行業慣例或個別協議而異，若以淨收入為基準，可能導致成員對成本的認定產生分歧，進而影響分配的公平性。

以總收入為基準則提供一個清晰、客觀的起點，讓成本提撥與分配原則可以在此基礎上進行協商與調整，確保整體制度的透明與公正。例如：

- **委託案情境**：外部客戶給付 100,000 元。某成員主張成本是 30,000 元（淨收 70,000 元）；另一成員主張成本是 20,000 元（淨收 80,000 元）。若以淨收入計算公司池提撥，會因成本認定不同而產生 10,000 元差異。
- **過案情境**：成員對外收 50,000 元，聲稱支付外包成本 15,000 元。難以驗證這 15,000 元的合理性。若改以總收入為基準計算 15% 服務費（7,500 元），則無需爭議成本定義。

以總收入為基準，確保所有計算「一目瞭然」。
:::

## 💼 委託案 Commisioned Work

### <small>PAY-02</small> 委託案定義與公司池提撥 {#pay-02}

1. 委託案係指外部委託人直接委託本社承接之案子；本社為直接承包人。
2. 委託案總收入中，應提撥至少{params.POOL_COMMISSION_RATE}至公司池（見 [POOL-01](/bylaws/07-pool/#pool-01)）。
3. 若有明確帶案並成功簽約者（最多以兩位帶案人為限），得自該案總收入中另提撥10%作為帶案獎金（在公司池{params.POOL_COMMISSION_RATE}之外）；分配方式由帶案人協議並向大會備查。
4. 其餘資金由參與成員協議分配，並遵守倍率限制（見 [PAY-07](/bylaws/06-pay-distribution/#pay-07)）。

### <small>PAY-03</small> 委託案盈餘與獎金 {#pay-03}

委託案分配後若仍有盈餘，得由參與該案之勞工老闆與外部夥伴共同決策作為獎金；仍須遵守倍率限制（見 [PAY-07](/bylaws/06-pay-distribution/#pay-07)）（除非大會另有決議）。

## 🌱 自籌案 Self-funded Project

### <small>PAY-04</small> 自籌案定義與回報方案 {#pay-04}

1. 自籌案係指本社成員自發發起之內部或外部專案，無外部委託人；資金來源為自籌、群眾贊助或本社內部投資。
2. 自籌計畫提案人若完成提案，則有權利獲得該計畫總收入之至少{params.POOL_COMPREHENSIVE_RATE}作為計畫與行政金。
3. 自籌案回報方案的估值原則同 [GRANT-01](/bylaws/08-grants-selfinitiated/#grant-01)，應考慮市場價格、成本價值、使用價值與對組織的貢獻度等因素，並由提案人提供估值依據與說明；大會可要求提供第三方估值或參考資料以確保估值的合理性與透明度。
4. 若自籌案的資金來源為本社勞工老闆們投資，最終案件的回收報酬應在扣除成本與計畫行政金後，根據投資比例優先分配給投資者；分配前須遵守倍率限制（見 [PAY-07](/bylaws/06-pay-distribution/#pay-07)），以確保公平性與內部協作的和諧。
5. 投資比例的計算方式應在投資時明確約定，並且在案件結束後進行核算與分配；投資比例並不影響決策權的分配，無視投資比例，所有參與的勞工老闆在案件的決策過程中仍享有平等的發言權與表決權。
6. 自籌案不強制先行提撥{params.POOL_GRANT_COMPREHENSIVE_RATE}現金，但須於案前提出公司回報方案（現金、器材、授權、場地、commons成果等），並以市場價值與實際現金價值估算、登錄與結案交付，回報方案應經勞工大會討論與記錄。

```
!!! 如果找其他人來工作？如果有的勞工老闆不想要參與投資，或是想要找外部夥伴來協助執行，該怎麼辦？
或如果勞工老闆想參與但不想投資怎麼辦？這些情況都很常見，應該要有彈性的處理方式。
```

## 💛 補助案 Grant/Subsidy

### <small>PAY-05</small> 補助案定義與綜合回收 {#pay-05}

1. 補助案係指申請政府或民間機構補助之案子；本社為直接申請人或承包人。
2. 每個補助案若有明確的主要撰寫計劃書人，並成功申請到計畫（不得超過二人），則可各自分配案子中的 5% 總收入作為帶案獎金。
3. 補助案若限制勞務費比例，應提出分配方案並確保公平；且至少應有綱先{params.POOL_GRANT_COMPREHENSIVE_RATE}價值由本社回收並共享（可為現金或可持續資產/commons成果）。估值原則同 [PAY-04](/bylaws/06-pay-distribution/#pay-04)。
4. 綜合回收方案應經勞工大會討論與記錄。

## 🔗 過案 Pass-through Work

### <small>PAY-06</small> 過案定義與行政服務費 {#pay-06}

1. 過案係指成員對外簽約承接之案子，本社僅提供對外簽約、開立發票、收款、帳務與行政承擔之服務；本社為行政支持方，不直接參與專案執行。
2. 過案總收入原則收取行政服務費{params.POOL_PASSTHROUGH_RATE}作為公司池收入。
3. 本條之{params.POOL_PASSTHROUGH_RATE}係「有別於」依法需繳納之營業稅/相關稅捐；稅捐依規定另行處理，不得將稅捐誤作公司池收入或行政服務費。
4. 如過案實質上涉及本社較高風險或工作量（合約責任、客訴、多人協作、使用本社資產/品牡、需PM等），大會得決議視同委託案，改按 [PAY-02](/bylaws/06-pay-distribution/#pay-02) 提撣公司池至少{params.POOL_COMMISSION_RATE}。

## 🔧 通用規則

### <small>PAY-07</small> 三倍原則：比較基準＝總額 {#pay-07}

:::note[為什麼需要三倍原則？]
避免參與專案的成員之間出現過大薪資差距，維持公平性。如果外部合作者領取是成員的十倍、百倍，就違背了合作社「公平勞動」的精神。我們認為三倍是可接受的彈性範圍，既不過於僵化，也保有空間考慮經驗、專門性差異。
:::

適用於委託案、自籌案與補助案：各位參與夥伴之總領取金額其他參與人之總領取金額相比，原則上不得差距超過三倍；比較基準為同一專案同一結算期間之總額。

### <small>PAY-07A</small> 三倍原則的適用範圍與例外 {#pay-07a}

工作坊、講座、短期專案、製作費等一次性合作或專案，因工作量與貢獻難以量化，且通常涉及外部專家或講者，得不受三倍原則限制；但須經大會討論並記錄決議，並參考本社內的最低薪標準與市場行情，以確保合理性與公平性。

### <small>PAY-08</small> 成本審批：兩位同意 {#pay-08}

任何案內成本（材料、器材、交通、外包等）須至少兩位勞工老闆同意（含提案人）始得支出；未事前寫入預算之新增成本亦同。憑證與核准紀錄必須存檔。（亦可參照 [PROJ-02](/bylaws/05-projects-conflicts/#proj-02)）

### <small>PAY-09</small> 報價底線與付款 {#pay-09}

1. 本社應制定最低可接受報價與工作條件（範圍、時程、修改輪次、責任邊界），並以此拒絕壓榨式委託。
2. 委託案與補助案原則採訂金或里程碑付款；逾期付款之停工與延後交付機制由細則規定。
3. 客戶違約造成損失原則由公司池風險準備承擔；若因未依流程簽約/收款等重大疏失造成，大會得決議補救或追究責任。
