## 📋 版本規範

我們使用 [語義化版本](https://semver.org/lang/zh-TW/) 管理章程版本：

- **主版本號 (Major)**：重大架構變更或不向後相容的修改
- **次版本號 (Minor)**：新增章節或重要條文
- **修訂號 (Patch)**：文字修正、澄清說明、錯字修正

### 範例

- `v1.0.0` → `v1.0.1`：修正錯字
- `v1.0.1` → `v1.1.0`：新增附註或補充條文
- `v1.1.0` → `v2.0.0`：修改核心治理機制
