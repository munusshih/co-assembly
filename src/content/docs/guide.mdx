本頁說明如何有效使用 CoAssembly 章程網站。

## 搜尋功能

點擊右上角的 🔍 搜尋圖示，即可搜尋全站內容。支援繁體中文搜尋。

:::tip[搜尋技巧]

- 搜尋專有名詞（如「勞工老闆」）可快速找到相關條文
- 搜尋條文編號（如「MEM-01」）可直接定位到該條文
- 使用 `/` 快捷鍵快速開啟搜尋
  :::

## 詞彙表連結

章程中的[專有名詞](/meta/glossary/)會以連結標示。點擊即可查看定義。

例如：

- [勞工老闆](/meta/glossary/#worker-owner)
- [準勞工老闆](/meta/glossary/#associate-worker-owner)
- [公司池](/meta/glossary/#pool)
- [共識決](/meta/glossary/#consensus)

## 跨章節參照

章程中使用自訂 ID 系統，讓您能輕鬆跳轉到相關條文：

- 站內連結格式：`[條文名稱](/bylaws/章節/#條文id)`
- 例如：[MEM-01](/bylaws/02-membership/#mem-01) 會連結到第二章的 MEM-01 條文

## 導航功能

- **側邊欄**：左側顯示完整章節結構
- **目錄**：右側顯示當前頁面的標題導航
- **上一頁/下一頁**：頁面底部可循序閱讀
- **編輯此頁**：每頁底部可在 GitHub 上直接編輯

## 版本控制

所有章程變更都記錄在 [變更紀錄](/meta/changelog/) 中。如需貢獻，請參閱 [如何貢獻](/meta/contributing/)。
