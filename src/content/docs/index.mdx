## 🌟 關於 CoAssembly

CoAssembly（共同集合設計合作社）是一個以協作、透明、共好為核心價值的組織。我們相信好的組織治理來自於清晰的規範與成員的共同參與。

本章程記錄了我們的運作原則、治理方式、財務制度與成員權利義務，並持續透過 GitHub 進行版本管理與公開修訂。
